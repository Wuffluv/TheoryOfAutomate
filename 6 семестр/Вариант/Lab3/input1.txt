% Пример кода на MATLAB
... Вычисление суммы элементов вектора

v = [1, 2, 3, 4, 5]; % Входной вектор


%{
Это многострочный комментарий.
Здесь мы выполняем итерацию по элементам вектора
и суммируем их значения
%}


sum = 0; % Инициализация суммы

for i = 1:length(v) % Итерация по элементам вектора
    sum = sum + v(i); % Накопление суммы
end

disp('Сумма элементов вектора:'); % Вывод результата
disp(sum);

% Конец программы
